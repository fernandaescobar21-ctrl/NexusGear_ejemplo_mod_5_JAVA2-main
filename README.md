# ⚡ Nexus Gear

Proyecto académico del Módulo 5 — Bootcamp Alkemy  
Tienda web de equipamiento gamer desarrollada con Java EE clásico.

## 🚀 Tecnologías usadas
- Java EE + Servlets + JSP
- Patrón MVC + DAO
- JDBC + MySQL
- JSTL + CSS
- Maven + Tomcat 10

## ▶️ Cómo ejecutar
1. Ejecutar `database/schema.sql` en MySQL
2. Configurar contraseña en `DBConexion.java`
3. Abrir en IntelliJ y configurar Tomcat 10
4. Ejecutar ▶️ y entrar a: `http://localhost:8080/NexusGearApp/`
